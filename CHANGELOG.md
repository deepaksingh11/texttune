# TextTune Changelog

## [Initial Release] - {PR_MERGE_DATE}

### Added
- Rewrite selected text from any macOS app using AI
- Support for OpenAI, Anthropic, and Groq providers
- 6 rewrite tones: Professional, Casual, Fix Grammar, Concise, Friendly, Formal
- Quick tone and provider switching via dedicated Raycast commands
- Paste rewritten text back to active app with Enter
- API keys stored securely in macOS Keychain
